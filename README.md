# 🕯️ Bayou Bazaar

> Small-batch, handmade candles from South Louisiana.

A demo storefront built with **Astro 5**, **WollyCMS**, and **Tailwind v4**, deployed to Cloudflare Pages.

## Products

- 🕯️ **Jar Candles** — Soy and blended jar candles in bayou-inspired scents (40–60 hr burn)
- 🏛️ **Pillar Candles** — Handcrafted beeswax pillars
- ✨ **Taper Candles** — Hand-dipped beeswax tapers, sold in sets
- 🌸 **Wax Melts** — Fragrance-packed wax melt cubes

## Stack

| Layer | Tech |
|-------|------|
| Frontend | Astro 5 (static) |
| CMS | WollyCMS + SQLite |
| Styles | Tailwind v4 |
| Deploy | Cloudflare Pages |
